# Hello_World_AI_App 🤖

My first AI app built with Python, Streamlit, and HuggingFace Transformers.

**What it does:** Type any sentence and get instant AI sentiment analysis - POSITIVE or NEGATIVE with confidence score.

**Tech used:** Python, Streamlit, PyTorch, Transformers

**Run locally:**
```bash
pip install streamlit transformers torch
